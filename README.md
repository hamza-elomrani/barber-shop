# barber shop

Responsive barber shop website built with Vite + React + TypeScript.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on Vercel

This project is configured for Vercel static deployment with SPA rewrites.

1. Import the repository into Vercel.
2. Use the default Node.js environment.
3. Build command: `npm run build`
4. Output directory: `dist`

The `vercel.json` file already handles route rewrites to `index.html` for client-side routing.
